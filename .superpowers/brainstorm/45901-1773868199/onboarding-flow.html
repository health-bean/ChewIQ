Design decisions:
• 3 steps (down from 5) — Welcome, Protocol, Ready
• Removed: Goals (ask in first chat), Tutorial (learn by doing), Sample Data (dev feature)
• Progress dots at top — clear position awareness
• "Skip" option on protocol — no forced choices for overwhelmed users
• "Ready" screen shows example prompts — teaches by example, not instructions
• All design system tokens — zero off-system colors
• Large touch targets, minimal text — brain fog friendly
