Component Library Gaps

You have 5 UI primitives. A real design system needs ~12-15. Which of these missing components should we build?

!

Dialog / Modal

Used in: StartReintroductionModal, confirmations, quick-add sheet. Currently each builds its own overlay/backdrop. Need one standardized component with focus trap, escape-to-close, and body scroll lock.

!

Select / Dropdown

Used in: Settings (protocol picker), admin pages. Currently raw <select> elements that look different from the styled Input component. Need a styled Select matching Input's aesthetic.

!

Textarea

Used in: Reflect page (notes), chat input. Currently hand-styled with inconsistent colors. Should match Input component's label/error pattern.

+

Tabs / Toggle Group

Used in: Chat mode toggle (Chat vs Quick Log), admin section switching. Currently inline ad-hoc styling. A reusable Tabs component would standardize all toggle UIs.

+

Toast / Notification

Used in: Save confirmations, error feedback, entry logged success. Currently using inline text that disappears. A toast system gives consistent, accessible feedback across all actions.

+

Empty State

Used in: Timeline (no entries), Insights (no data), Reintroductions (none active). Currently each page builds its own. Need a consistent template: icon + title + description + optional CTA.

○

Avatar

Used in: Chat messages (user vs AI), header user area. Would add personality — initials badge for user, leaf/plant icon for AI assistant.

○

Progress Bar

Used in: Protocol phase progress, reintroduction day tracking, onboarding steps. Currently uses custom divs with inline width. Need a proper animated Progress component.

○

Skeleton Loader

Used in: Every page that fetches data. Currently shows a centered Spinner. Skeleton loaders (animated placeholder shapes) feel faster and more polished.

🔴 Red (!) = Critical gaps — these cause visual inconsistency right now
🟡 Yellow (+) = High value — missing patterns that would elevate the product feel
⚪ Gray (○) = Nice to have — polish items that round out the system
